# Image credits

Attribution log for every external image used in the quiz database. Inline
SVG authored within this repository (traffic signs, symbols) needs no entry —
it is original work created for this project and carries no third-party
licence.

## How to add an entry

When you introduce an external image (`imgType: "img"`):

1. Confirm the licence explicitly permits reuse (PD / CC / open data).
2. Add a row above with the source URL and licence (e.g. `CC BY-SA 4.0`).
3. Put the same short credit in the question item's `imgCredit` field.

Never add an image whose licence you cannot verify.
