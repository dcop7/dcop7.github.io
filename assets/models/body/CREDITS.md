# 3D body models — credits & licence

All models below come from **NIH 3D** (https://3d.nih.gov) and are licensed
**Creative Commons Attribution 4.0 (CC BY 4.0)**. They were downloaded, decimated
and meshopt-compressed for the web (kept locally so the explorer works offline).
No model is altered in meaning; only polygon count / file size were reduced.
